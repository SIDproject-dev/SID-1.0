# SID-1.0
Search and Identify Drone

An autonomous quadcopter that can map and navigate an indoor space and identify persons in the space.

<br/>
<br/>

Hardware:
  
  NVIDIA Jetson TK1
  
  Pixhawk
  
  Microsoft Xbox 360 Kinect
